# Spam Email Classifier

This project builds a machine learning model to classify emails as Spam or Not Spam.

## Technologies Used
- Python
- Scikit-learn
- TF-IDF
- Naive Bayes

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python spam_classifier.py

## Output
The model prints accuracy and classification report.
